L'Effet Parallaxe

Tutoriel : Comment créer cet effet ?

Le parallaxe consiste à déplacer l'image de fond plus lentement que le texte pour créer de la profondeur.

1. Structure HTML

On utilise des sections avec une classe commune pour l'effet et des classes uniques pour les images.

2. Le CSS Magique

La propriété indispensable est background-attachment: fixed. Elle bloque l'image par rapport à l'écran pendant que vous scrollez.

Propriété Fixed

/* Exemple de code CSS */
.parallax {
    background-attachment: fixed;
    background-size: cover;
    height: 600px;
}
